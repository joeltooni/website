Resources

A curated list of tools, libraries, and resources I find useful in my work and studies.

Computer Vision

  • Camera Model Visualization - Interactive tool for understanding camera geometry and projection.
  • Camera Calibration Visualization - Learn about lens distortion and calibration techniques.
  • OpenCV - The most widely used library for computer vision tasks.
  • YOLO (You Only Look Once) - Real-time object detection system.
  • Detectron2 - Facebook AI's library for object detection and segmentation.
  • MMDetection - OpenMMLab's detection toolbox for object detection and instance segmentation.
  • Albumentations - Fast image augmentation library.

Robotics

  • ROS (Robot Operating System) - The most popular framework for robot software development.
  • OpenCV - A library for computer vision tasks.
  • Gazebo - A simulation tool for testing robot designs.

Machine Learning

  • PyTorch - My favorite deep learning framework.
  • TensorFlow - The other major deep learning framework.
  • Scikit-learn - A library for traditional machine learning algorithms.
  • Keras - A high-level neural networks API, running on top of TensorFlow.

Research

  • arXiv - Preprint repository for research papers.
  • Google Scholar - Academic search engine.
  • Zotero - Reference management tool.
  • Overleaf - Online LaTeX editor.

Development Tools

  • Visual Studio Code - My primary code editor.
  • Git - Version control system.
  • Docker - Containerization platform.
  • GitHub - Code hosting and collaboration.
